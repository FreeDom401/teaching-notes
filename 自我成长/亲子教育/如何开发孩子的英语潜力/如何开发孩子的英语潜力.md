# 0. 英语启蒙不是抢跑

1. 语言学习要先听说，后读写

2. 语言学习要沉浸式，不要翻译，用英语学习英语

3. 学语言要短时、高频

   * 每天10-15分钟的空闲时间学习、游戏，和孩子一起进行英文对话。

   * 碎片化的时间利用方式。

4. 词汇积累比口语更重要

上小学之前，50%要在听说上，30%放在从听到阅读上，20%打下写的基础。

# 1. 儿童英语启蒙的目标

 学龄前英语启蒙的目标：

* 听力方面，有1000个听力词汇的积累，能够听得懂，可以执行指令、回答问题等。
* 内容方面，能就九大主题进行沟通和表达。
  * 食物和基本需求
  * 喜好和意愿
  * 数字和物品
  * 地点和工作
  * 动物及其习性
  * 服装
  * 玩具和周边环境
  * 生活事件
  * 日常用语和指令
* 阅读方面，上学前能够积累约450本分级阅读和英文绘本的听读量，积累大约300个阅读词汇，也就是看见了能认识，知道什么意思；
* 写的方面，能用大概150个左右的单词，进行简单写作。

# 2. 家庭游戏，创造日常英语环境

在了解孩子的认知发展水平基础上，词汇串联跟句子拓展这样的小游戏，可以随时在家玩，出门玩，看见什么说什么，就可以达到很好的英语启蒙效果。

需要注意的是，在指导孩子学英语上，一不要翻，而不要背。所有的练习小游戏，一定要反复滚动、不断浮现。另外，带你的孩子做英语游戏，英语水平高不高根本不重要，一颗温柔陪伴孩子的新才是最重要的。

一次学习10-20分钟，每周3-5次。

# 3. 英文读物怎么挑

1. 首先，基本不能指望孩子通过看英文动画片学英语

2. 图文字典是个宝贝，它可以帮助孩子有效避开母语，直接跟图片进入英文，有利于培养英语思维

   * 小学前
     * 《学乐儿童图画字典》
     * 《My First Dictionary》
   * 小学后
     * 《Children‘s IIIustrated Dictionary》
     * 《麦克米伦儿童字典》

3. 分级阅读材料会比绘本优先级高，大量高频重复的单词，才是孩子学英语的突破口。

   * Reading A to Z
   * 培生幼儿启蒙英语Level A+B 

4. 英文绘本选经典

   三年时间，大概270本分级阅读+180本的英文原版绘本。

   * 第一年，150本左右的分级阅读
   * 第二年，读70本分级阅读加80本英文绘本
   * 第三年，读50本分级阅读加100本英文绘本

# 4. 英语阅读四步法

* 第一步：借图解意

  先看图不看字，过程中用全英文跟孩子沟通，这一步是为了通过图片理解意思

* 第二步：你读他听，音意结合

  你读孩子听，读的时候，要比正常语速慢一点，这一步是为了让孩子把英文与具体的意思对应上

* 第三步：指指点点，初步理解

  指指点点，就是用手把重要的信息圈出来，这一步是为了帮助孩子进一步理解书里面的意思

* 第四步：说一说，理解深度

  说一说，就是引导孩子说一说这个故事，这一步为了让孩子深入理解这个故事。

# 5. 怎么组合使用分级书单

* 每周两次坚持读
* 每月前三周读新书，最后一周读已经读过的书。每半年的第六个月，就在前五个月读过的30本书挑一些来读。
* 第六个月，对前面读过的30本书按照主题来分类重读
